# Skill: Screen Vision

## Description
Captures screenshots and describes screen content using LLaVA via Ollama.

## Trigger
Queries containing: "screen", "screenshot", "what do you see", "look at", "describe what".

## Constraints
- Max resolution: 1280x720
- Rate limit: 1 capture per 5 seconds
- Uses screenshot-desktop for capture
- LLaVA via Ollama for description

## Phase
P3 — Awareness
