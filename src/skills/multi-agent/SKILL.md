# Skill: Multi-Agent System

## Description
Parallel specialist sub-agents for complex tasks. Research, Code, File, and Analysis agents work concurrently.

## Trigger
Complex tasks that benefit from parallel specialist processing.

## Agents
- **Research Agent**: Information gathering
- **Code Agent**: Code writing and debugging
- **File Agent**: File system operations
- **Analysis Agent**: Code review and architecture

## Phase
P4 — Autonomy
