# Skill: RAG Engine

## Description
Retrieval-Augmented Generation engine for grounded answers from documents. Uses local embeddings via @xenova/transformers.

## Trigger
Queries that reference documents, files, or knowledge base content.

## Features
- Local embeddings (no API calls)
- Document chunking and indexing
- Similarity search for relevant context
- Grounded answer generation (reduces hallucination)
