# Skill: Proactive Cron

## Description
Scheduled autonomous tasks using node-cron. JARVIS can perform actions on a schedule without user prompting.

## Trigger
User configures cron jobs via Gateway UI or CLI.

## Constraints
- Anti-drift limit: 3 messages per hour max
- Uses node-cron for scheduling
- All scheduled tasks logged to episodic memory

## Phase
P3 — Awareness
